# 🚀 サーバー起動方法

## ローカルサーバーの起動

以下のコマンドでサーバーを起動してください：

```bash
# 1. プロジェクトディレクトリに移動
cd "/Users/komiyakouhei/Desktop/名称未設定フォルダ 6"

# 2. 仮想環境を有効化
source venv/bin/activate

# 3. サーバーを起動
python run.py
```

## 🌐 アクセスURL

サーバー起動後、以下のURLにブラウザでアクセスしてください：

### **http://localhost:8000**

## 📝 注意事項

- サーバーを停止する場合は、ターミナルで `Ctrl+C` を押してください
- 初回起動時は、`.env`ファイルにAPI認証情報を設定する必要があります
- ポート8000が使用されている場合は、環境変数`PORT`で別のポートを指定できます

## ✅ 動作確認

1. ブラウザで `http://localhost:8000` にアクセス
2. ページが表示されれば成功です
3. 「API接続テスト」ボタンで各APIの接続をテストできます
