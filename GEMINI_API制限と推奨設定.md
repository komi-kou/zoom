# Gemini API制限と推奨設定

## 📊 Gemini APIモデルの比較

### Gemini 2.5 Pro
- **無料枠**: 1日100回まで
- **RPM（1分あたりリクエスト数）**: 5回
- **料金**: 入力 $1.25/100万トークン、出力 $10.00/100万トークン
- **特徴**: 高精度、長文要約に強い
- **推奨**: 精度重視で1日100回以下で十分な場合

### Gemini 2.5 Flash
- **無料枠**: 1日1,500回まで
- **RPM**: 10,000回
- **料金**: 入力 $0.075/100万トークン、出力 $0.30/100万トークン（2.5 Proの約1/16）
- **特徴**: 高速、コスト効率が非常に良い、精度も良好
- **推奨**: 自動処理に最適！コストとパフォーマンスのバランスが良い

### Gemini 1.5 Pro
- **無料枠**: 1日1,500回まで
- **RPM**: 5回
- **料金**: 入力 $1.25/100万トークン、出力 $5.00/100万トークン
- **特徴**: 高精度、長文対応
- **推奨**: 精度重視でコストを抑えたい場合

### Gemini 1.5 Flash
- **無料枠**: 1日1,500回まで
- **RPM**: 15回
- **料金**: 入力 $0.075/100万トークン、出力 $0.30/100万トークン
- **特徴**: 最もコスト効率が良い、速度も速い
- **推奨**: コスト重視、大量処理が必要な場合

## 🎯 自動処理での推奨設定

### 推奨1: Gemini 2.5 Flash（最推奨）
```bash
GEMINI_MODEL_NAME=gemini-2.5-flash-exp
```
- **理由**: 
  - 1日1,500回まで無料
  - コストが2.5 Proの約1/16と非常に安い
  - 処理速度が速い
  - 精度も十分高い
- **適している場合**: 自動処理で大量に使う場合

### 推奨2: Gemini 1.5 Flash（コスト最優先）
```bash
GEMINI_MODEL_NAME=gemini-1.5-flash
```
- **理由**:
  - 最もコスト効率が良い
  - 1日1,500回まで無料
  - 安定したモデル
- **適している場合**: コストを最小限に抑えたい場合

### 推奨3: Gemini 2.5 Pro（精度最優先）
```bash
GEMINI_MODEL_NAME=gemini-2.5-pro
```
- **理由**:
  - 最高精度
  - 長文要約に強い
- **注意**: 1日100回の制限あり、コストが高い
- **適している場合**: 精度が最重要で、1日100回以下で十分な場合

## 💡 自動処理での使用回数の目安

### 1日あたりの使用回数
- **少規模**: 1日10-50回 → Gemini 2.5 Proでも可
- **中規模**: 1日50-200回 → Gemini 2.5 Flash推奨
- **大規模**: 1日200回以上 → Gemini 1.5 Flash推奨

### 1回の処理コスト（目安）
- **Gemini 2.5 Pro**: 約$0.01-0.05/回（録画の長さによる）
- **Gemini 2.5 Flash**: 約$0.001-0.003/回（約1/16）
- **Gemini 1.5 Flash**: 約$0.001-0.003/回（約1/16）

## ⚙️ 設定方法

`.env`ファイルでモデルを変更：

```bash
# 推奨: Gemini 2.5 Flash（自動処理に最適）
GEMINI_MODEL_NAME=gemini-2.5-flash-exp

# または Gemini 1.5 Flash（コスト最優先）
# GEMINI_MODEL_NAME=gemini-1.5-flash

# または Gemini 2.5 Pro（精度最優先、1日100回制限）
# GEMINI_MODEL_NAME=gemini-2.5-pro
```

**自動処理で大量に使う場合の推奨**:
